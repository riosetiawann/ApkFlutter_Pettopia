## Instalation

cole aplikasi menggunakan kode berikut:
'''
git clone https://github.com/riosetiawann/ApkFlutter_Pettopia.git
'''

Masuk ke direktori project
...
cd ApkFlutter_Pettopia
...

install dependencies
...
flutter pub get
...

start aplikasi
...
flutter run
...
